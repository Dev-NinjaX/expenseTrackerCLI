# Expense Tracker (CLI)

This is a basic command-line expense tracker built in Java.  
I created this project to practice core Java concepts and understand how to structure a small application.

## What it does

- Add expenses  
- View all saved expenses  
- Update expenses  
- Show a monthly summary  
- Delete expenses  
- Save data to a text file so it persists after closing the program  

## Why I built this

I wanted to get comfortable with:

- Using ArrayList  
- Handling user input with Scanner  
- Reading and writing files  
- Organizing code into multiple classes  

## How to run

1. Clone the repository  
2. Go into the project folder  
3. Compile: javac ExpenseTracker.java
4. Run
